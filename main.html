Política de Privacidade — Mensagens Evangélicas

Última atualização: 01 de março de 2026

1. Introdução

O aplicativo Mensagens Evangélicas ("aplicativo", "nós") é um serviço gratuito que oferece mensagens e conteúdo evangélico, pedidos de oração comunitários e assistente de chat com inteligência artificial. Esta Política de Privacidade descreve como coletamos, usamos, armazenamos e protegemos suas informações pessoais ao utilizar o aplicativo.

Ao utilizar nosso aplicativo, você concorda com as práticas descritas neste documento. Se não concordar com qualquer parte desta política, pedimos que não utilize o aplicativo.

2. Dados Coletados

2.1 Dados de Conta (fornecidos pelo usuário)

Ao realizar login com o Google, coletamos automaticamente:

  • Nome de exibição
  • Endereço de e-mail
  • URL da foto de perfil (fornecida pela conta Google)
  • Identificador único do usuário (UID do Firebase Authentication)

Esses dados são sincronizados com nossa base de dados (Cloud Firestore) na conta do usuário.

2.2 Conteúdo Criado pelo Usuário

  • Pedidos de oração: título, descrição e comentários publicados na seção de orações comunitárias. O usuário pode optar por publicar anonimamente, caso em que nenhum nome ou foto é armazenado publicamente.
  • Interações: contagem de curtidas/orações realizadas, favoritos e compartilhamentos (contadores numéricos associados à conta).
  • Chat com IA: mensagens enviadas ao assistente de inteligência artificial durante a sessão. As conversas do chat não são persistidas em nossos servidores — elas existem apenas durante a sessão ativa no dispositivo.

2.3 Dados Técnicos

Não coletamos diretamente dados de dispositivo, localização, câmera, microfone ou qualquer outro sensor. O acesso à internet é necessário para o funcionamento do aplicativo.

4. Compartilhamento de Dados com Terceiros

Para operar o aplicativo, compartilhamos dados com os seguintes provedores de serviço:

4.1 Google Firebase (Google LLC)

  • Firebase Authentication: gerencia o login e a sessão do usuário.
  • Cloud Firestore: banco de dados onde são armazenados perfis de usuário, pedidos de oração e comentários.

Política de privacidade: https://policies.google.com/privacy

4.2 Google Sign-In (Google LLC)

Utilizado para autenticação via conta Google. As permissões solicitadas são: profile (nome e foto) e email.

Política de privacidade: https://policies.google.com/privacy

4.3 OpenRouter.ai

As mensagens enviadas ao chat de IA são transmitidas à API da OpenRouter, que as encaminha a modelos de linguagem de terceiros (como GPT, Claude, Gemini) para gerar respostas. Não enviamos dados de identificação pessoal (nome, e-mail, foto) a este serviço — apenas o conteúdo das mensagens da conversa.

Política de privacidade: https://openrouter.ai/privacy

4.4 Google Fonts (Google LLC)

Utilizado apenas para carregamento de fontes tipográficas. Nenhum dado pessoal é transmitido.

Não vendemos, alugamos ou cedemos seus dados pessoais a terceiros para fins comerciais.

5. Armazenamento e Segurança dos Dados

  • Os dados são armazenados nos servidores do Google Firebase, com infraestrutura segura gerenciada pelo Google.
  • O acesso ao banco de dados é controlado por regras de segurança do Firestore, garantindo que cada usuário acesse apenas seus próprios dados privados.
  • Utilizamos conexões criptografadas (HTTPS/TLS) para todas as comunicações de rede.
  • Embora adotemos medidas razoáveis de segurança, nenhum sistema é completamente inviolável. Não podemos garantir segurança absoluta.

6. Retenção de Dados

  • Seus dados de conta e conteúdo são mantidos enquanto sua conta estiver ativa.
  • Caso solicite a exclusão da conta, removeremos seus dados pessoais do nosso banco de dados em até 30 dias, exceto onde a retenção seja exigida por lei.
  • Pedidos de oração publicados anonimamente permanecem no sistema sem vínculo identificável com sua conta.

7. Seus Direitos

De acordo com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018), você tem direito a:

  • Acessar os dados que temos sobre você.
  • Corrigir dados incompletos, inexatos ou desatualizados (disponível na tela de perfil do aplicativo).
  • Excluir seus dados pessoais e sua conta.
  • Revogar o consentimento para uso dos dados.
  • Portabilidade dos seus dados, mediante solicitação.
  • Informação sobre com quem compartilhamos seus dados.

Para exercer qualquer um desses direitos, entre em contato pelo e-mail indicado na Seção 10.

8. Dados de Menores de Idade

Este aplicativo não é destinado a crianças menores de 13 anos. Não coletamos intencionalmente dados de menores. Se você acreditar que uma criança forneceu informações pessoais ao aplicativo sem consentimento dos responsáveis, entre em contato conosco para que possamos removê-las.

9. Alterações a Esta Política

Podemos atualizar esta Política de Privacidade periodicamente. Quando houver alterações significativas, notificaremos por meio de aviso no aplicativo ou atualizando a data de "Última atualização" no topo deste documento. Recomendamos que você revise esta política regularmente.
